## What's pinned

- **[job-match-radar](https://github.com/ozlar34/job-match-radar)**: End-to-end n8n pipeline. Scrapes LinkedIn + 6 ATS endpoints, recovers missing job descriptions via JSON-LD, scores listings against a personal rubric in Claude Code, dedupes, and emails a digest.
- **[Sidekick](https://github.com/ozlar34/Sidekick)**: Native macOS edge-anchored notes panel. Hotkey-summoned, plain `.md` storage, hybrid markdown editor. SwiftUI + AppKit. Replaces SideNotes for a folder-on-disk workflow.
- **[StanzaApp](https://github.com/ozlar34/StanzaApp)**: Menu-bar timer for sit/stand desk cycles. Native macOS, no Electron. Built because every existing pomodoro app gets the cycle wrong.
- **[claude-code-skills](https://github.com/ozlar34/claude-code-skills)**: A handful of Claude Code skills I built for my own stack — capture/handoff utilities, Obsidian + Notion + NotebookLM workflows, 3D printing. 5 are runnable; 4 are walkthroughs of patterns too tightly coupled to my private setup to publish source.
