## What I build outside work

#### Automating my own workflow

- **[job-match-radar](https://github.com/ozlar34/job-match-radar)** — end-to-end n8n + Supabase pipeline. Scrapes LinkedIn + ATS endpoints, recovers missing job descriptions via JSON-LD, scores listings against a personal rubric in Claude Code, dedupes, and emails a weekly digest.
- **[claude-code-skills](https://github.com/ozlar34/claude-code-skills)** — Claude Code skills I built for my own stack: capture/handoff utilities, Obsidian + Notion + NotebookLM workflows, 3D printing. 5 run out of the box; 4 are walkthroughs of patterns too tightly coupled to my private setup to publish source.

#### Building (native macOS)

- **[Sidekick](https://github.com/ozlar34/Sidekick)** — edge-anchored notes panel. Hotkey-summoned, plain `.md` storage, hybrid markdown editor. SwiftUI + AppKit. Replaces SideNotes for a folder-on-disk workflow.
- **[StanzaApp](https://github.com/ozlar34/StanzaApp)** — menu-bar timer for sit/stand desk cycles. Native macOS, no Electron. Built because every existing pomodoro app gets the cycle wrong.
